## ðŸš€ How to Use v1.0.0

### Installation
```bash
git clone https://github.com/devstd2211/edison-smart-levels-trading-bot.git
cd edison-smart-levels-trading-bot
npm install
npm run build
```

### Running the Bot
```bash
# Single strategy mode
npm start

# Development mode
npm run dev

# With web dashboard
npm run dev:full

# Web dashboard only
npm run dev:web
```

### Backtesting
```bash
# Quick backtest
npm run backtest:xrp --limit 100

# Optimize parameters
npm run backtest:optimize --symbol BTCUSDT

# Walk-forward analysis
npm run backtest:walkforward
```

### Testing
```bash
# Run all tests
npm test

# With coverage
npm test -- --coverage
```
